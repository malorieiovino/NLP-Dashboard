## 📌 About  
This is an interactive **NLP Dashboard** built using **Streamlit**. It allows users to analyze text using:  
- 🔍 **Word Frequency Analysis** (Word Cloud & Top Words)  
- 🏷 **Named Entity Recognition (NER)**  
- 😊 **Sentiment Analysis**  
- ✍️ **Part-of-Speech (POS) Tagging**  
- 📑 **Topic Modeling** (LDA)
